## Key Capabilities

* Drop-In Replacement for OpenTofu or Terraform
* Enhanced Developer Experience through more ergonomic syntax and error reporting
* Global Shared Provider Caching
* Scripting API
* Version Manager for OpenTofu and Terraform

## Quick Start

AETF can be used as a drop in replacement, just open your project and run:

```bash
aetf plan --no-transpile --legacy-passthrough
```

if you want to leverage the new features provided by AETF, rename your files to ".aetf" and don't pass the `--no-transpile` flag.

For more information check our [Documentation](https://docs.aetf.dev/).
